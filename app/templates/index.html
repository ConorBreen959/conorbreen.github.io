Dashboards Of Interest

Here are some dashboards I've built in my spare time. I've made these as a demonstration of skill, some might be useful but mostly they're here so I can show off.

Primer Design

Extracts flanking sequences of variable distance for a given species given a single BP position.

Copernicus

Calculates conjunctions of a selection of heavenly bodies.

SRA Toolkit

Possibly downloads genetic data from an SRA accession ID, but I haven't decided what it will do.

SAD Optimism

Provides sunrise and sunset times for today's data or a date selection.

Flexible Visualisation Tool

Upload any dataset (in a tsv file, in long format, with column headers) and visualise the data with a customisable graph.
